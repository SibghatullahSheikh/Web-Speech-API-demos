Demo: Speech recognition based on Web Speech API

(Only tested on last version of Chrome browser).

Push "space" key or to start the speech recognition.

Log:

Demos using the speech recognition.

You try to speak: "Light on". If the bulb is off, you should see lamp switching on.
You try to speak: "Light off". If the bulb is on, you should see lamp switching off.

You try to speak: "Play". The video should start playing.
You try to speak: "Stop". The video should stop playing.
